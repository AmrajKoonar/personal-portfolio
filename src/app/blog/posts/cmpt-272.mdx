## Course Overview

CMPT 272 provides an introduction to client-side web development, with a focus on building interactive, responsive, and structured web applications using modern web technologies.

## Key Topics Covered

- **HTML and Markup Languages**: Page structure, semantic elements, forms, and organizing content for the web
- **CSS and Styling**: Layouts, selectors, responsive design, visual styling, and user interface presentation
- **JavaScript Programming**: Dynamic behaviour, event handling, DOM manipulation, and client-side logic
- **TypeScript Fundamentals**: Static typing, safer JavaScript development, and improving code reliability
- **Front-end Frameworks**: Introduction to popular frameworks used to build modern web applications
- **Web Architecture**: Understanding how browsers, clients, servers, and web applications communicate
- **REST API Usage**: Invoking web services using basic API calls and working with data from existing REST APIs

## Skills Developed

- Building structured web pages using HTML and semantic markup
- Styling responsive and usable interfaces with CSS
- Creating dynamic client-side behaviour using JavaScript
- Applying TypeScript to improve code quality and maintainability
- Understanding the role of front-end frameworks in modern web development
- Working with APIs to fetch and display data in web applications
- Understanding the basic architecture of web applications and client-server communication

## Projects and Assignments

- Web page development exercises using HTML and CSS
- Client-side programming tasks using JavaScript and TypeScript
- Interactive user interface assignments involving DOM manipulation and event handling
- Responsive layout and styling exercises
- Front-end framework based development tasks
- API integration exercises using existing REST APIs
- Assignments focused on applying web development concepts to practical client-side applications

## Learning Outcomes

This course developed a strong foundation in client-side web development, including the ability to structure, style, and program interactive web applications. It strengthened my understanding of HTML, CSS, JavaScript, TypeScript, front-end frameworks, web architecture, and API usage in modern web development.
