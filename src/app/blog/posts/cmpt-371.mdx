## Course Overview

CMPT 371 provides an introduction to data communications and computer networking, with a focus on how modern networks are designed, operated, and analyzed.

## Key Topics Covered

- **Data Communication Fundamentals**: Data types, transmission rates, transmission media, and basic communication concepts
- **Network Architectures**: Local area networks, wide area networks, protocol layering, and Internet architecture
- **Application Layer Protocols**: Principles of network applications including HTTP, DNS, and socket programming
- **Transport Layer Protocols**: UDP, TCP, reliable data transfer, flow control, and congestion control
- **Network Layer and Routing**: IP addressing, forwarding, routing algorithms, and protocols such as OSPF, RIP, and BGP
- **Link Layer and LANs**: Multiple access protocols, error detection and correction, Ethernet, bridges, and switches
- **Network Performance Analysis**: Evaluating networks under different traffic loads and channel error rates

## Skills Developed

- Understanding of computer networking models and layered architectures
- Analysis of communication protocols across application, transport, network, and link layers
- Socket programming and practical network application development
- Routing, forwarding, and IP addressing concepts
- Performance evaluation of networks under varying load and error conditions
- Understanding of LAN technologies, Ethernet, switches, and error control methods

## Projects and Assignments

- Network protocol analysis and problem solving
- Socket programming and client server communication exercises
- Transport layer reliability, flow control, and congestion control analysis
- Routing and IP addressing exercises
- Link layer and LAN performance analysis
- Assignments focused on applying networking concepts to real communication systems

## Learning Outcomes

This course developed a strong foundation in data communications and networking, including the structure of the Internet, major networking protocols, and the performance tradeoffs involved in designing reliable and efficient communication systems.
