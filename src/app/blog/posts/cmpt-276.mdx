## Course Overview

CMPT 276 provides a comprehensive introduction to software engineering principles and practices.

## Key Topics Covered

- **Git Version Control**: Learned essential Git commands and workflows for collaborative development
- **Design Patterns**: Studied common software design patterns and their applications
- **Software Development Life Cycle (SDLC)**: Explored various SDLC methodologies including Agile and Waterfall
- **Software Architecture**: Understanding of software architecture principles and best practices
- **Team Collaboration**: Experience working in development teams using modern tools and practices

## Skills Developed

- Version control with Git and GitHub
- Understanding of software design patterns
- Knowledge of SDLC methodologies
- Team collaboration and project management
- Software architecture and design principles

## Projects and Assignments

- Collaborative software development projects
- Implementation of design patterns in practical applications
- Version control workflow exercises
- Software architecture design assignments

## Learning Outcomes

This course provided a solid foundation in software engineering principles, preparing for more advanced development work and team-based projects.
