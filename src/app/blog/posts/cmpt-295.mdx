## Course Overview

CMPT 295 provides an introduction to computer systems, with a focus on how programs are represented, executed, and optimized at the machine level.

## Key Topics Covered

- **Computer Organization**: CPU structure, memory, instruction execution, and the relationship between hardware and software
- **Data Representation**: Binary numbers, signed and unsigned integers, floating point representation, bitwise operations, and data formats
- **Machine Level Programming**: Assembly language, registers, instructions, memory addressing, control flow, and function calls
- **Systems Programming in C**: Pointers, arrays, structs, memory layout, compilation, linking, and debugging
- **Instruction Sets and Processor Design**: Instruction set architecture concepts and how processors execute instructions
- **Memory Hierarchy**: Caches, locality, virtual memory, and how memory access patterns affect program performance
- **Performance Analysis**: Measuring and improving program performance using compiler optimization and profiling tools

## Skills Developed

- Understanding of how high level code is translated into machine level instructions
- Ability to read, write, and debug assembly language programs
- Stronger understanding of C programming, memory layout, pointers, and data representation
- Analysis of how hardware design affects program execution and performance
- Use of debugging and profiling tools such as GDB, Valgrind, and perf
- Understanding of caches, locality, and memory hierarchy performance
- Ability to reason about instruction execution, control flow, and low level program behavior

## Projects and Assignments

- C programming exercises focused on pointers, memory, and data representation
- Assembly language labs involving arithmetic, control flow, functions, and stack behavior
- Debugging exercises using GDB and related command line tools
- Performance measurement tasks using compiler optimization and profiling tools
- Cache and memory hierarchy experiments analyzing memory access patterns
- Branch prediction and low level performance analysis assignments
- Processor and instruction set related exercises connecting hardware design with software execution

## Learning Outcomes

This course developed a strong foundation in computer systems by connecting high level programming concepts to the hardware that executes them. It strengthened my understanding of C, assembly language, memory, caches, instruction sets, and performance analysis, while showing how low level system behavior affects the correctness and efficiency of real programs.
