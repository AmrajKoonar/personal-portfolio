## Overview

Built a cybersecurity tool that detects phishing attempts using AI-powered email and file analysis. 

**Live link:** [Deep Phishing](https://deep-phishing.vercel.app/)

## Key Features

- **AI-Powered Analysis**: Utilized OpenAI API to analyze email content and detect potential phishing attempts
- **File Analysis**: Implemented file scanning capabilities to detect malicious content
- **User-Friendly Interface**: Created an intuitive web interface using Next.js and React
- **Modern UI**: Styled with Tailwind CSS and Shadcn/ui components for a professional look
- **Automated Testing**: Implemented comprehensive testing with automated deployment on Vercel

## Technologies Used

- **Next.js**: For server-side rendering and optimal performance
- **React**: For building interactive user interfaces
- **Tailwind CSS**: For responsive and modern styling
- **Shadcn/ui**: For pre-built, accessible UI components
- **OpenAI API**: For AI-powered content analysis
- **JavaScript**: For client-side functionality
- **Vercel**: For deployment and hosting

## Development Process

The project involved extensive research into cybersecurity threats and phishing detection methods. The development process included:

- Research and analysis of common phishing techniques
- Integration with OpenAI API for content analysis
- Implementation of file upload and scanning functionality
- User interface design and development
- Comprehensive testing and deployment

## Outcome

The DeepPhishing tool successfully detects various types of phishing attempts with high accuracy. The project demonstrates expertise in AI integration, cybersecurity concepts, and modern web development practices.
