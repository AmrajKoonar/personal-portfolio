## Overview

Built a Python-powered Twitter bot that delivers real-time weather alerts, road conditions, parking updates, and live webcam screenshots for SFU Burnaby campus.

**Live link:** [SFU StormSage](https://x.com/sfulivewebcams)

## Key Features

- **Real-Time Weather Alerts**: Posted live updates about weather, road conditions, and parking lot status at SFU Burnaby campus
- **Live Webcam Integration**: Included webcam screenshots with updates to provide better visual awareness of current campus conditions
- **Web Scraping**: Used BeautifulSoup to extract live text and image data from the SFU Road Conditions page
- **Twitter/X Automation**: Integrated Tweepy with the Twitter API to automatically publish updates
- **Campus-Focused Alerts**: Designed the bot specifically to help SFU students and staff stay informed about changing campus conditions
- **Auto-Updating System**: Built automated logic to continuously fetch new data and post updates when conditions change

## Technologies Used

- **Python**: For scripting the bot, scraping data, and automating update logic
- **BeautifulSoup**: For parsing and extracting information from the SFU Road Conditions page
- **Requests**: For fetching live webpage data and image resources
- **Tweepy**: For integrating with the Twitter API and posting automated updates
- **Twitter/X API**: For publishing weather alerts and webcam updates to the live bot account
- **Web Scraping**: For collecting structured and unstructured road condition data
- **Environment Variables**: For securely managing API keys and access tokens

## Development Process

The project focused on building an automated information bot to keep the SFU community updated about live weather and campus access conditions. The development process included:

- Researching SFU road condition and webcam data sources
- Building Python scraping logic to collect text updates and webcam image data
- Parsing live road, weather, and parking information from the SFU Security Road Conditions page
- Integrating Tweepy with the Twitter API to automate posting
- Creating a secure environment variable setup for Twitter API credentials
- Designing update logic so the bot could fetch new data and publish alerts as conditions evolved
- Testing the bot locally to verify scraping, image handling, and Twitter posting functionality

## Outcome

SFU StormSage provides SFU students and staff with a convenient way to receive live Burnaby campus weather, road, parking, and webcam updates through Twitter/X. The project demonstrates experience with Python automation, web scraping, API integration, live data processing, and practical campus-focused software development.
