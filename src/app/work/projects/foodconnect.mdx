## Overview

Developed an online donation platform connecting food donors and recipients, featuring a marketplace for food donations.

## Key Features

- **Donation Platform**: Created a system for food donors to list available items
- **Marketplace**: Implemented a marketplace interface for browsing and claiming donations
- **User Authentication**: Built secure user registration and login system
- **Database Management**: Designed and implemented PostgreSQL database for data persistence
- **Cloud Deployment**: Deployed on AWS for scalability and reliability
- **Real-time Updates**: Implemented real-time notifications for donation status

## Technologies Used

- **React**: For building the frontend user interface
- **TypeScript**: For type-safe development and better code quality
- **Tailwind CSS**: For responsive and modern styling
- **Express.js**: For building the backend API
- **PostgreSQL**: For data persistence and management
- **AWS**: For cloud deployment and hosting
- **Socket.io**: For real-time communication features

## Development Process

The project focused on creating a sustainable solution for food waste reduction through community engagement. The development process included:

- Market research and user needs analysis
- Database design and schema planning
- Frontend and backend development
- User interface design and user experience optimization
- Testing and deployment on AWS infrastructure

## Outcome

FoodConnect successfully connects food donors with recipients, reducing food waste and helping communities. The platform demonstrates full-stack development skills, database design, and cloud deployment expertise.
