## Overview

Developed an exclusive marketplace platform for Simon Fraser University students to buy, sell, and trade items safely within the SFU community. *(IN PROGRESS)*

## Key Features

- **Secure Authentication**: Implemented JWT-based authentication with SFU email validation
- **SFU Student Verification**: Restricted account creation to users with valid SFU email addresses
- **Marketplace Listings**: Built functionality to create, edit, delete, and browse marketplace posts
- **Advanced Filtering**: Created category-based filtering and real-time search for marketplace listings
- **User Dashboard**: Developed protected dashboard for users to manage listings and view marketplace activity
- **Contact Sellers**: Added functionality for students to contact sellers about marketplace items
- **Animated Landing Page**: Designed a long-form homepage with GSAP-powered scroll animations
- **Responsive Design**: Created a modern and mobile-friendly interface using Tailwind CSS

## Technologies Used

- **Next.js**: For building the full-stack web application and API routes
- **React**: For creating interactive and reusable user interface components
- **JavaScript**: For client-side functionality, authentication flow, and API integration
- **Supabase**: For database management and PostgreSQL storage
- **PostgreSQL**: For storing user accounts, verification status, and marketplace posts
- **JWT Authentication**: For secure user sessions and protected routes
- **Tailwind CSS**: For responsive styling and modern UI design
- **GSAP**: For scroll-based animations and interactive landing page effects

## Development Process

The project focused on creating a trusted student-only marketplace for the SFU community. The development process included:

- Designing a marketplace platform tailored specifically for SFU students
- Building account creation and login functionality with SFU email validation
- Creating Supabase database tables for user accounts and marketplace posts
- Implementing protected routes using JWT authentication and cookie-based sessions
- Developing backend API routes for account management and post CRUD operations
- Building a dashboard where users can browse, filter, create, edit, and delete listings
- Designing a responsive interface using Tailwind CSS and SFU-inspired branding
- Adding GSAP animations to create a more polished and engaging landing page

## Outcome

SFU Marketplace provides Simon Fraser University students with a dedicated platform to buy, sell, and trade within a trusted campus community. The project demonstrates experience with full-stack web development, authentication, database design, API development, responsive UI design, and modern deployment workflows.
