## Overview

Developed a study tool for students to generate custom AI quizzes from uploaded notes using OpenAI API.

## Key Features

- **AI Quiz Generation**: Utilized OpenAI API to create custom quizzes from uploaded notes
- **File Upload**: Implemented secure file upload system for various document formats
- **Customizable Quizzes**: Created options for different question types and difficulty levels
- **Progress Tracking**: Built system to track learning progress and quiz performance
- **User-Friendly Interface**: Designed intuitive interface for easy note upload and quiz creation
- **Responsive Design**: Created mobile-friendly interface using Tailwind CSS

## Technologies Used

- **Next.js**: For server-side rendering and optimal performance
- **React**: For building interactive user interfaces
- **JavaScript**: For client-side functionality and API integration
- **Tailwind CSS**: For responsive and modern styling
- **OpenAI API**: For AI-powered quiz generation from text content
- **File Processing**: For handling various document formats

## Development Process

The project focused on creating an educational tool to help students study more effectively. The development process included:

- Research into educational technology and study methods
- Integration with OpenAI API for content analysis and quiz generation
- Implementation of file upload and processing functionality
- User interface design for optimal learning experience
- Testing with various document formats and content types

## Outcome

ThinkForge provides students with an innovative way to study by converting their notes into personalized quizzes. The project demonstrates expertise in AI integration, educational technology, and modern web development practices.
